
# Yabbl

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)


A realtime chat application made with React


## Features

- Light/dark mode toggle
- Profile customization
- Chatroom creation
- Direct messages

## Run Locally

Clone the project

```bash
  git clone https://github.com/dougmab/yabbl
```

Go to the project directory

```bash
  cd yabbl
```

Install dependencies

```bash
  npm install
  # or
  yarn install
```

Start the server

```bash
  npm run dev
  # or
  yarn run dev
```


## Environment Variables

To run this project, you will need to add the following environment variable to your .env file

`VITE_API_URL` - URL of your API (https://example.com)


## API

Your can check the application's API repository [here](https://github.com/dougmab/yabbl-api).
## Tech Stack

**Client:** Vite, React, TailwindCSS

**Server:** Java, Spring
